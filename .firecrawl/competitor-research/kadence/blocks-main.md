Is Kadence Blocks free?

Kadence Blocks is free and open-source, you can use it however you want.

We also offer a Paid Pro Extension which provides added features, extra blocks, and extra prebuilt content. You can view [pricing here](https://www.kadencewp.com/kadence-blocks/pro/).

Will this slow down my website?

Kadence Blocks was built with intelligent code that make it as lightweight as possible. This means it only loads what you are using on a page by page basis.

For example, if you have a page where you are not using any Kadence Block then nothing from Kadence Blocks will load. Or if you have a page where you are only using our Advanced Button block then only the advanced button block styles will load.

The intelligent load means that if you just want to use one block from our plugin you don’t have to load all the other css and javascript as extras.
