# SGS Forms Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Build a complete form system (13 blocks + processing engine) into the SGS Blocks plugin, replacing Fluent Forms / SureForms for all client sites.

**Architecture:** All form blocks are dynamic (render.php) to avoid deprecation headaches and enable server-injected nonces, unique field IDs, and validation error rendering. The `sgs/form` wrapper uses the WordPress Interactivity API for multi-step navigation, form submission via fetch, and tile selection. Field validation uses HTML5 attributes client-side, with full server-side validation as the security boundary.

**Tech Stack:** WordPress 6.9+, @wordpress/scripts, @wordpress/interactivity, PHP 8.0+, REST API, dbDelta for table creation, N8N webhooks for notifications.

**Phase 1 Scope (this plan):**
- 13 blocks: `sgs/form`, `sgs/form-step`, 10 field blocks (text, email, phone, textarea, select, radio, checkbox, tiles, file, consent), `sgs/form-review`
- Processing engine: REST submit + upload endpoints, DB table, server-side validation, N8N webhook
- Interactivity API viewScriptModule on form wrapper for multi-step + submission
- Form CSS for all elements

**Phase 2 Deferred (NOT this plan):**
- Stripe payment integration
- Address with postcode lookup
- Date, number, hidden fields
- Conditional logic JS
- Admin submissions view in wp-admin
- GDPR export/erasure hooks
- CSV export

### Step 3: Create the form processor class

`includes/forms/class-form-processor.php`:

```php
<?php
namespace SGS\Blocks\Forms;

defined( 'ABSPATH' ) || exit;

final class Form_Processor {

    /**
     * Process a form submission.
     *
     * @param string $form_id     Form identifier.
     * @param array  $fields      Submitted field data (key => value).
     * @param array  $file_ids    Attachment IDs from file uploads.
     * @return array|\WP_Error    Submission result or error.
     */
    public static function process( string $form_id, array $fields, array $file_ids = array() ) {
        $sanitised = self::sanitise_fields( $fields );

        $submission_id = self::store_submission( $form_id, $sanitised, $file_ids );

        if ( is_wp_error( $submission_id ) ) {
            return $submission_id;
        }

        self::fire_webhook( $form_id, $submission_id, $sanitised, $file_ids );

        return array(
            'submission_id' => $submission_id,
            'success'       => true,
        );
    }

    private static function sanitise_fields( array $fields ): array {
        $sanitised = array();

        foreach ( $fields as $key => $value ) {
            $clean_key = sanitize_key( $key );

            if ( is_array( $value ) ) {
                $sanitised[ $clean_key ] = array_map( 'sanitize_text_field', $value );
            } elseif ( str_contains( $clean_key, 'email' ) ) {
                $sanitised[ $clean_key ] = sanitize_email( $value );
            } elseif ( str_contains( $clean_key, 'url' ) || str_contains( $clean_key, 'website' ) ) {
                $sanitised[ $clean_key ] = esc_url_raw( $value );
            } else {
                $sanitised[ $clean_key ] = sanitize_text_field( $value );
            }
        }

        return $sanitised;
    }

    private static function store_submission( string $form_id, array $fields, array $file_ids ): int|\WP_Error {
        global $wpdb;

        $table = $wpdb->prefix . 'sgs_form_submissions';

        $result = $wpdb->insert(
            $table,
            array(
                'form_id'    => sanitize_text_field( $form_id ),
                'data'       => wp_json_encode( $fields ),
                'files'      => ! empty( $file_ids ) ? wp_json_encode( $file_ids ) : null,
                'ip_address' => self::get_client_ip(),
                'user_agent' => isset( $_SERVER['HTTP_USER_AGENT'] )
                    ? sanitize_text_field( wp_unslash( $_SERVER['HTTP_USER_AGENT'] ) )
                    : '',
                'user_id'    => get_current_user_id() ?: null,
                'status'     => 'new',
                'created_at' => current_time( 'mysql', true ),
            ),
            array( '%s', '%s', '%s', '%s', '%s', '%d', '%s', '%s' )
        );

        if ( false === $result ) {
            return new \WP_Error(
                'sgs_form_db_error',
                __( 'Failed to save submission.', 'sgs-blocks' ),
                array( 'status' => 500 )
            );
        }

        return (int) $wpdb->insert_id;
    }

    private static function fire_webhook( string $form_id, int $submission_id, array $fields, array $file_ids ): void {
        $webhook_url = apply_filters( 'sgs_form_webhook_url', '', $form_id );

        if ( empty( $webhook_url ) ) {
            return;
        }

        $file_data = array();
        foreach ( $file_ids as $attachment_id ) {
            $file_data[] = array(
                'id'   => $attachment_id,
                'name' => get_the_title( $attachment_id ),
                'url'  => wp_get_attachment_url( $attachment_id ),
            );
        }

        $payload = array(
            'form_id'       => $form_id,
            'submission_id' => $submission_id,
            'submitted_at'  => gmdate( 'c' ),
            'site_url'      => home_url(),
            'fields'        => $fields,
            'files'         => $file_data,
        );

        wp_remote_post( $webhook_url, array(
            'body'      => wp_json_encode( $payload ),
            'headers'   => array( 'Content-Type' => 'application/json' ),
            'timeout'   => 0.01,
            'blocking'  => false,
            'sslverify' => true,
        ) );
    }

    private static function get_client_ip(): string {
        $ip = '';

        if ( ! empty( $_SERVER['HTTP_CLIENT_IP'] ) ) {
            $ip = sanitize_text_field( wp_unslash( $_SERVER['HTTP_CLIENT_IP'] ) );
        } elseif ( ! empty( $_SERVER['HTTP_X_FORWARDED_FOR'] ) ) {
            $parts = explode( ',', sanitize_text_field( wp_unslash( $_SERVER['HTTP_X_FORWARDED_FOR'] ) ) );
            $ip    = trim( $parts[0] );
        } elseif ( ! empty( $_SERVER['REMOTE_ADDR'] ) ) {
            $ip = sanitize_text_field( wp_unslash( $_SERVER['REMOTE_ADDR'] ) );
        }

        return filter_var( $ip, FILTER_VALIDATE_IP ) ? $ip : '';
    }
}
```

### Step 4: Create the file upload handler

`includes/forms/class-form-upload.php`:

```php
<?php
namespace SGS\Blocks\Forms;

defined( 'ABSPATH' ) || exit;

final class Form_Upload {

    public static function handle( \WP_REST_Request $request ) {
        $files = $request->get_file_params();

        if ( empty( $files['file'] ) ) {
            return new \WP_Error(
                'sgs_form_no_file',
                __( 'No file was uploaded.', 'sgs-blocks' ),
                array( 'status' => 400 )
            );
        }

        $file = $files['file'];

        $max_size = apply_filters( 'sgs_form_max_upload_size', 10 * MB_IN_BYTES );
        if ( $file['size'] > $max_size ) {
            return new \WP_Error(
                'sgs_form_file_too_large',
                sprintf(
                    __( 'File exceeds the maximum size of %s.', 'sgs-blocks' ),
                    size_format( $max_size )
                ),
                array( 'status' => 400 )
            );
        }

        $allowed = apply_filters(
            'sgs_form_allowed_mime_types',
            array(
                'jpg|jpeg|jpe' => 'image/jpeg',
                'png'          => 'image/png',
                'gif'          => 'image/gif',
                'webp'         => 'image/webp',
                'pdf'          => 'application/pdf',
            )
        );

        $filetype = wp_check_filetype( $file['name'], $allowed );
        if ( ! $filetype['type'] ) {
            return new \WP_Error(
                'sgs_form_invalid_type',
                __( 'This file type is not allowed.', 'sgs-blocks' ),
                array( 'status' => 400 )
            );
        }

        if ( ! function_exists( 'wp_handle_upload' ) ) {
            require_once ABSPATH . 'wp-admin/includes/file.php';
        }
        if ( ! function_exists( 'wp_generate_attachment_metadata' ) ) {
            require_once ABSPATH . 'wp-admin/includes/image.php';
        }

        $upload = wp_handle_upload( $file, array(
            'test_form' => false,
            'mimes'     => $allowed,
        ) );

        if ( isset( $upload['error'] ) ) {
            return new \WP_Error(
                'sgs_form_upload_error',
                $upload['error'],
                array( 'status' => 500 )
            );
        }

        $attachment_id = wp_insert_attachment(
            array(
                'post_mime_type' => $upload['type'],
                'post_title'     => sanitize_file_name( pathinfo( $file['name'], PATHINFO_FILENAME ) ),
                'post_content'   => '',
                'post_status'    => 'private',
            ),
            $upload['file']
        );

        if ( is_wp_error( $attachment_id ) ) {
            return $attachment_id;
        }

        wp_update_attachment_metadata(
            $attachment_id,
            wp_generate_attachment_metadata( $attachment_id, $upload['file'] )
        );

        return array(
            'id'   => $attachment_id,
            'name' => $file['name'],
            'url'  => wp_get_attachment_url( $attachment_id ),
            'size' => $file['size'],
        );
    }
}
```

### Step 5: Create the REST API class

`includes/forms/class-form-rest-api.php`:

```php
<?php
namespace SGS\Blocks\Forms;

defined( 'ABSPATH' ) || exit;

final class Form_REST_API {

    public static function register(): void {
        add_action( 'rest_api_init', array( static::class, 'register_routes' ) );
    }

    public static function register_routes(): void {
        $namespace = 'sgs-forms/v1';

        register_rest_route( $namespace, '/submit', array(
            'methods'             => \WP_REST_Server::CREATABLE,
            'callback'            => array( static::class, 'handle_submit' ),
            'permission_callback' => '__return_true',
            'args'                => array(
                'formId' => array(
                    'type'              => 'string',
                    'required'          => true,
                    'sanitize_callback' => 'sanitize_text_field',
                ),
                'fields' => array(
                    'type'     => 'object',
                    'required' => true,
                ),
                'fileIds' => array(
                    'type'    => 'array',
                    'default' => array(),
                    'items'   => array( 'type' => 'integer' ),
                ),
                'honeypot' => array(
                    'type'    => 'string',
                    'default' => '',
                ),
            ),
        ) );

        register_rest_route( $namespace, '/upload', array(
            'methods'             => \WP_REST_Server::CREATABLE,
            'callback'            => array( static::class, 'handle_upload' ),
            'permission_callback' => '__return_true',
        ) );
    }

    public static function handle_submit( \WP_REST_Request $request ): \WP_REST_Response|\WP_Error {
        // Honeypot check — return fake success to not reveal the trap.
        $honeypot = $request->get_param( 'honeypot' );
        if ( ! empty( $honeypot ) ) {
            return rest_ensure_response( array(
                'success' => true,
                'message' => __( 'Thank you for your submission.', 'sgs-blocks' ),
            ) );
        }

        // Rate limiting.
        $ip        = self::get_client_ip();
        $form_id   = $request->get_param( 'formId' );
        $rate_key  = 'sgs_form_rate_' . md5( $ip . $form_id );
        $rate_count = (int) get_transient( $rate_key );

        if ( $rate_count >= 5 ) {
            return new \WP_Error(
                'sgs_form_rate_limited',
                __( 'Too many submissions. Please try again later.', 'sgs-blocks' ),
                array( 'status' => 429 )
            );
        }

        set_transient( $rate_key, $rate_count + 1, HOUR_IN_SECONDS );

        $fields   = $request->get_param( 'fields' );
        $file_ids = $request->get_param( 'fileIds' );

        $result = Form_Processor::process( $form_id, $fields, $file_ids );

        if ( is_wp_error( $result ) ) {
            return $result;
        }

        return rest_ensure_response( array(
            'success'       => true,
            'message'       => __( 'Thank you for your submission.', 'sgs-blocks' ),
            'submissionId'  => $result['submission_id'],
        ) );
    }

    public static function handle_upload( \WP_REST_Request $request ): \WP_REST_Response|\WP_Error {
        $result = Form_Upload::handle( $request );

        if ( is_wp_error( $result ) ) {
            return $result;
        }

        return rest_ensure_response( $result );
    }

    private static function get_client_ip(): string {
        if ( ! empty( $_SERVER['REMOTE_ADDR'] ) ) {
            $ip = sanitize_text_field( wp_unslash( $_SERVER['REMOTE_ADDR'] ) );
            return filter_var( $ip, FILTER_VALIDATE_IP ) ? $ip : '';
        }
        return '';
    }
}
```

### Important: Step visibility in view.js

The view.js must handle step visibility. When a form has multiple `.sgs-form-step` children, on initialisation it hides all steps except the first. The `nextStep`/`prevStep` actions update `context.currentStep` and toggle the `hidden` attribute on step elements.

Use safe DOM manipulation (no innerHTML):

```javascript
// In the review population function, use document.createElement and textContent:
const dt = document.createElement( 'dt' );
dt.textContent = entry.label;
const dd = document.createElement( 'dd' );
dd.textContent = entry.value;
reviewList.appendChild( dt );
reviewList.appendChild( dd );
```

## Task 11: Form CSS

**Files:**
- `plugins/sgs-blocks/src/blocks/form/style.css`
- `plugins/sgs-blocks/src/blocks/form/editor.css`

### Key CSS rules

1. **Field grid layout**: `.sgs-form__inner` and `.sgs-form-step` use CSS grid with `grid-template-columns: 1fr 1fr`. Width classes: `--full` spans both columns, `--half` spans 1, `--third` spans 1 (3-column achieved by nesting). Mobile: single column.

2. **Inputs**: 44px min-height (WCAG touch target), theme token border colour, 2px focus outline with primary colour.

3. **Tile grid**: CSS grid with configurable columns, accent border + background on `:has(input:checked)`, visually hidden inputs.

4. **File upload zone**: Dashed border, hover state, progress/preview sections.

5. **Progress bar**: Flex container for step indicators, coloured number badges for active/completed states.

6. **Buttons**: Primary/accent/success variants using `:not([style*="background-color"])` pattern for CSS fallbacks that yield to inline custom colours.

7. **Success/error messages**: Alert styling with appropriate colours.

8. **Honeypot**: `position: absolute; left: -9999px` (not `display:none` — bots check for that).

9. **Review list**: `<dl>` styled as 2-column grid (label + value).

10. **Radio/checkbox options**: Flex column layout, 44px min-height touch targets, accent colour on selection.
