# Proposal: Snooza Chair Website — Ophir Solutions

## Summary

A modern, fast WordPress website for the Snooza Chair featuring an interactive 3D product configurator with augmented reality — letting customers see every colour, every accessory, and the chair in their actual room before buying.

Built with the SGS WordPress Framework by Small Giants Studio.

## What You Get

### 1. Interactive 3D Product Configurator
- Customers spin the Snooza Chair in 3D from any angle
- Switch between all 6 colours instantly
- Toggle accessories on/off (tray, pommel, leg rest, headrest)
- Select size with automatic price update
- **AR:** Point your phone at the floor and see the chair in your room — no app needed

### 2. Online Ordering
- Configure → add to cart → checkout
- Secure payment via Stripe, PayPal, or Square
- Order confirmation with exact configuration saved
- No transaction fees on top of standard payment processing

### 3. Modern Website
- Fast, mobile-first WordPress site
- Built on the SGS Framework (custom block theme)
- SEO-optimised, accessible (WCAG 2.2 AA)
- Easy to update — no coding required
