## Design principles to enforce

1. **Trust before conversion** — every section earns trust before asking for action
2. **Premium but approachable** — generous whitespace, polished hover states, no aggressive design
3. **Respect user time** — clear hierarchy, scannable, obvious next steps
4. **Mobile-first** — works at 375px without compromise
5. **WCAG 2.2 AA** — text ≥ 4.5:1, large text ≥ 3:1, UI components ≥ 3:1
6. **Touch targets** — minimum 44 × 44px for interactive elements
7. **Heritage feel** — premium independent business, not corporate or discount

## Hard exclusions

- No bare hex colours in styles (must reference palette tokens)
- No greyscale on brand-strip logos (full colour throughout)
- No `--undefined` class suffixes (recently fixed in testimonial; verify no others)
- No `300msms` or other malformed CSS values
- No animation on header / footer / page-wrapper / structural containers
- No hover-scale on display-only blocks (counters, strips)
- No pure-white surface on Indus (warm-tints OK, but Indus uses #FFFFFF for clinical look)
- All animations must respect `prefers-reduced-motion`
