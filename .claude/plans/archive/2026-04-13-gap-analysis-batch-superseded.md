# Spec 05 — SUPERSEDED

**This spec is no longer the path forward.** Kept as a pointer.

## What this used to be

A batch gap-analysis pass over 17 design sub-skills (adapt, audit, bolder, clarify, colourise, critique, delight, distill, extract, harden, interactive-design, normalize, onboard, optimise, polish, quieter, teach-impeccable) using the OLD criteria: completeness, clarity, routing_accuracy, robustness, security, negative_routing, exemplar_quality, ecosystem_awareness — scored 0-5 each.

## Why it's superseded

The 2026-04-27/28 sessions replaced this approach with **end-goal rubrics**. Each skill now gets a `references/end-goal-rubric.md` with end-result anchors (5/5 = gold output, 1/5 = broken), behaviour-anchored statements per criterion, mandatory Never-Do lists, and optional Lens-6 anchors.

22 of these rubrics shipped on 2026-04-27/28, including 11 of the 17 skills targeted here:
colourise, bolder, quieter, normalize, polish, distill, delight, extract, harden, adapt, critique.

The remaining 6 (audit, clarify, interactive-design, onboard, optimise, teach-impeccable) are Phase 2b targets, evaluated under the same new framework with a triage filter.

## What to do instead

For any new gap-analysis or rubric work:
1. Read `~/.claude/skills/gap-analysis/references/end-goal-rubric.md` for the methodology
2. Read the per-skill rubrics under `~/.agents/skills/<skill>/references/end-goal-rubric.md`
3. See `~/.openclaw/.claude/subprojects/ssb/specs/2026-04-27-optimisation-toolkit-design.md` for the full 5-phase tooling-rebuild plan
