# sgs/media — FS-4 visual-diff (Method-2 converter-lift, Phase B)

## Change
`plugins/sgs-blocks/src/blocks/media/style.css` — the `.wp-block-sgs-media img, .sgs-media__img` rule gained `width:100%` + `min-width:0` (keeping `max-width:100%` + `height:auto` + `object-fit`). Block-owned classes only; no client/section selector (R-22-9 universal). Fixes the 0×0 collapse when an `<img>` is a direct grid/flex child whose track sizing + `min-width:auto` interaction zeroes the box.
