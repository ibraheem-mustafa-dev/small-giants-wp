## Migration summary
- Supports skip-serialised: spacing, color, __experimentalBorder (radius), typography → scoped `.uid` CSS.
- Border took the quote route: native width/colour/style dropped → custom `borderWidth` object + `borderColour`/`borderStyle` scalars; radius stays WP-native + tier objects.
- Box-object attrs: paddingTablet/Mobile, marginTablet/Mobile, borderRadiusTablet/Mobile, borderWidth. `box_family` seeded centrally.
- Collapsed the repeated per-item inline `style="color:…"` (same value on every `<li>`) into ONE scoped
  `.uid .sgs-icon-list__icon` / `__text` rule; array data untouched.
- Device tiers 1023/767. uid = class (anchor-safe). No version bump / deprecations (D293).
