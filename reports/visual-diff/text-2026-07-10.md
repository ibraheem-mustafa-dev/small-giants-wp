# sgs/text — D303 per-instance scope normalisation (LANDED)

**Verdict: PASS.** text scoped its per-instance responsive CSS at ID specificity (`#$anchor`) — the
element carried the anchor only as an `id`. D303 moves the scope to class level
(`.wp-block-sgs-text.$anchor`, 0,2,0) and adds the anchor token as a CLASS on the wrapper (the
`id="…"` is kept for operator anchors / linking) so the selector matches — the appended
`sgsCustomCss` residual (0,2,0) can override it by source order, no ID.
