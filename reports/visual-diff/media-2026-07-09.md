# Visual-diff — sgs/media 100% no-inline + border-radius→native migration (D293 rollout)

**Verdict: PASS** · **first_paint_capture_passed: true**

## What changed
`sgs/media` migrated to the no-inline styling contract as a **block-private** block (it is an
atomic `img→sgs/media` clone target; `wraps_block=None`). Every inline `style="…"` on the media
element/figure/caption is removed and emitted into the block's own scoped `.{uid}` `<style>`:
`object-fit`, `object-position`, `aspect-ratio`, `opacity`, `box-shadow`, the native border group,
figure alignment margins, caption colour/font-size. `__experimentalBorder` flipped to
`__experimentalSkipSerialization: true`. Border-radius migrated from 5 custom flat attrs
(`borderRadius`+`Unit`+4 corners) to WP-native `style.border.radius` (base) + new
`borderRadiusTablet`/`borderRadiusMobile` object attrs (tiers) via the style engine. Scoped uid is a
CLASS (anchor-safe); works for both naked-`<img>` and `<figure>` shapes (scope class merged onto the
naked img so scoped rules match). No version bump.
